# Songwriting Craft — Full Reference (from `songwriting-and-ai-music` skill)

## Song Structure (Pick One or Invent Your Own)

```
ABABCB  Verse/Chorus/Verse/Chorus/Bridge/Chorus    (most pop/rock)
AABA    Verse/Verse/Bridge/Verse (refrain-based)    (jazz standards, ballads)
ABAB    Verse/Chorus alternating                    (simple, direct)
AAA     Verse/Verse/Verse (strophic, no chorus)     (folk, storytelling)
```

Six building blocks: Intro, Verse, Pre-Chorus, Chorus, Bridge, Outro.

## Rhyme Types (Tight to Loose)

Perfect (lean/mean), Slant (prove/love), Sight (rough/dough), Identical (light/light), Assonance (feet/eat), Consonance (boat/debt), Para (ender/bitter).

## AI Music Prompting (Suno)

### Basic prompt format
```
[Genre]: electronic, synthwave, retro
[Style]: cinematic, energetic, driving beat
[Key]: Cm
[Tempo]: 128 BPM
[Instruments]: synth pads, arpeggiators, drum machine
[Mood]: nostalgic, futuristic
[Vocals]: instrumental (or specify vocal style)
```

### Lyric formatting for Suno
```
[Intro] (spoken, whispered)
[Verse 1]
...
[Chorus]
...
[Bridge]
...
[Outro]
```

## Key principle: Art breaks rules on purpose. Use what serves the song.
